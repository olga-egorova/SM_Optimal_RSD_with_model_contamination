
Each folder in this Supplementary material contains code and output corresponding to each section of the manuscript:


- "CRD example": search for compound optimal designs in the completely randomized example, Section 3.4 (Figures 2-4), Appendix A (Figures 7-9) and B (Figure 10)

- "Generalized criteria": search for Generalized D- and L-optimal designs in the CRD example, Section 3.4 (Figure 5)

- "CRD Pareto": search for Pareto frontier of designs in the completely randomized example, Section 3.4, Appendix C

- "Case study": search for compound optimal designs in the blocked case study example, Section 4.1 (Figure 6)


Within each folder, there is a .Readme file
